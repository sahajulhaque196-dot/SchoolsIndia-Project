How to Find Any School in India Using UDISE Code

By Sahajul  ·  Tezpur, Assam  ·  January 10, 2025  ·  9 min read

A few months ago, I received a message from a reader in rural Odisha. Her daughter had just passed Class 8 and the family was trying to find out whether the government school in the next block — one they had heard offered science up to Class 12 — was actually operational or had been closed after the pandemic. Nobody in the village knew for certain. The school's name kept coming up in conversations but nobody had any real information.

That kind of uncertainty is surprisingly common across India. We have over sixteen lakh schools in this country, and yet for millions of families — particularly in rural areas — finding reliable, verified information about a specific school remains genuinely difficult. The UDISE code system exists precisely to solve this problem. But very few people outside education administration circles know how it works or how to use it.

In this guide, I am going to explain everything you need to know about UDISE codes: what they mean, how they are structured, and how you can use them to find accurate, official information about any school in India.

What Is UDISE?

UDISE stands for Unified District Information System for Education. It is a nationwide database maintained by the Ministry of Education, Government of India, that records detailed information about every school in the country — from the largest private schools in Mumbai to the smallest single-teacher primary schools in remote tribal villages.

The system was initially developed in the 1990s and has been continuously updated and expanded. The current version, UDISE+, is a web-based platform that collects data annually from schools across all states and union territories. Every school that submits data to UDISE receives a unique 11-digit code called the UDISE code. This code is the school's official identity in the national education database.

Think of it the way you might think of an Aadhaar number — not for a person, but for a school. Just as your Aadhaar uniquely identifies you in government records, the UDISE code uniquely identifies each school in the education database. No two schools share the same code.

How Is the UDISE Code Structured?

The 11-digit UDISE code is not random. It is built from the school's administrative geography, and each set of digits carries specific meaning. Here is how it breaks down:

The first two digits represent the state or union territory. For example, 09 represents Uttar Pradesh, 27 represents Maharashtra, 35 represents Andaman and Nicobar Islands. There are codes for all 36 states and UTs.

The next two digits (digits 3 and 4) represent the district within that state. Each district has been assigned a unique two-digit code within its state.

The next three digits (digits 5, 6, and 7) represent the block or tehsil within the district.

The final four digits are the unique school identifier within that block.

So if you see a UDISE code like 27050100101, you can read the first two digits (27) and know it is a school in Maharashtra. The next two digits (05) tell you the district, and so on. This structure makes the code both a unique identifier and a geographic marker — something that comes in very useful when you are looking at large datasets.

Why Does the UDISE Code Matter to Parents?

For most parents, the UDISE code is probably not something they think about in daily life. But there are several situations where knowing a school's UDISE code — or being able to look it up — is genuinely useful.

First, it is used in government scholarship applications. Many central and state government scholarship schemes require students to enter their school's UDISE code as part of the application. If you have the wrong code — or if the school's details in UDISE are outdated — your application can be rejected on technical grounds.

Second, it is used in mid-day meal programme monitoring. The government tracks meal distribution at the school level using UDISE codes. If a school's code is not properly registered or has not submitted data recently, it can affect the school's eligibility for various central schemes.

Third, it helps parents verify whether a school is officially recognised. A school that appears in the UDISE database is one that has submitted data to the government and has some level of official recognition. While UDISE listing does not guarantee everything about a school, its absence from the database can be a warning sign.

How to Find a School's UDISE Code

There are several ways to find a school's UDISE code. The most direct method is to ask the school itself — the UDISE code should appear on official correspondence from the school, on government-issued recognition certificates, and on the school's noticeboard in many states.

If you cannot get the information directly from the school, you can use the search feature on this website. Schoolspedia.in allows you to search by school name, state, district, or block. Once you find the school, its UDISE code is prominently displayed on its page.

You can also search the official UDISE+ portal at udiseplus.gov.in, though the official interface can be less intuitive for general users.

What Information Can You Find With a UDISE Code?

Once you have a school's UDISE code, you can look up a comprehensive set of information. On Schoolspedia.in, every school page shows the school's official name, complete address down to the village and cluster level, management type, school category (which classes it offers), whether it is co-educational or single-gender, and its current operational status.

The page also shows the broader demographic context of the district the school is in — literacy rates, population figures, the number of schools and post offices in the district, and other data that helps you understand the educational landscape of the area.

A Final Word

India's education system is vast and often opaque. The UDISE system, for all its limitations, is one of the most serious attempts to bring transparency and data-driven governance to that system. Understanding how to use it — even at the most basic level — gives ordinary citizens a meaningful tool for making informed decisions about education.
